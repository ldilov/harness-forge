# Review Template

## Change Summary

- schema changes
- data movement changes
- application rollout dependencies

## Risk Verdict

- level: low, medium, high, or blocked
- why that level is justified

## Compatibility Window

- old app against new schema
- new app before backfill completes
- contract conditions for cleanup

## Rollout Plan

- deploy order
- backfill strategy
- observability or validation steps
- rollback or forward-fix plan
