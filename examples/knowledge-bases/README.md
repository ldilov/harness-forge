# Seeded Knowledge Examples

The seeded language packs ship their scenario examples in place under
`knowledge-bases/seeded/<language>/examples/`.

## Example families

- TypeScript: API, React library, Next.js app, monorepo contracts
- Java: Spring Boot, event consumer, multi-module backend, domain library
- .NET: ASP.NET Core, worker, shared domain, CLI automation
- Lua: Neovim, OpenResty, LÖVE, embedded automation
- PowerShell: script module, CI automation, remoting, bootstrap tooling
