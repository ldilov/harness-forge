# Template Selection Cheatsheet

## By task

- Patch comparison: `certificates/patch-equivalence-certificate.md`
- Fault localization: `certificates/fault-localization-certificate.md`
- Code semantics answer: `certificates/code-question-answering-certificate.md`
- Change safety review: `certificates/change-safety-certificate.md`
- Fast disciplined investigation: `contracts/semiformal-core-contract.md` + `logs/semiformal-exploration-log.md`

## By risk mode

- Lite: core contract + short exploration log
- Standard: core contract + exploration log + one certificate
- Deep: full exploration + task-specific certificate + unresolved assumptions + explicit recommendation posture

## Escalation rule

If the opposite answer remains plausible, escalate instead of compressing uncertainty.
