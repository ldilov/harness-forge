# Adoption Paths

## Lite

Start here when the question is narrower and the risk is lower.

Use:

- core contract
- enough cited evidence to support the answer

## Standard

Use this when the answer depends on real trace depth or multi-file behavior.

Add:

- exploration log
- relevant ledgers
- specialized certificate where appropriate

## Deep

Use this when merge safety, multiple hypotheses, or hidden semantics make the
task high consequence.

Add:

- full exploration record
- specialized certificate
- stronger reviewer and counterexample pressure
