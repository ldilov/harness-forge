# Pre-merge Semiformal Checklist

## Framing

- [ ] Intended behavior is explicit.
- [ ] Preserved and changed behaviors are separated.
- [ ] In-scope boundaries are explicit.

## Evidence Quality

- [ ] Critical path tracing is present.
- [ ] Evidence items include source locations.
- [ ] Reviewer spot-check can replay a decisive claim.

## Counterexample Pressure

- [ ] Counterexample (or no-counterexample argument) is documented.
- [ ] Alternative recommendation check is documented.
- [ ] Hidden semantics assumptions are surfaced.

## Decision Quality

- [ ] Recommendation is one of allowed outcomes.
- [ ] Unresolved assumptions are visible.
- [ ] Conditions/blockers are explicit when not unconditional.
