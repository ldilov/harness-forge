# Certificate Review Checklist

- [ ] Certificate type matches task shape
- [ ] Task statement and scope are explicit
- [ ] Material claims link to evidence locations
- [ ] Alternative hypothesis was explicitly checked
- [ ] Counterexample handling is present (or no-counterexample argument)
- [ ] Formal conclusion does not introduce unsupported claims
- [ ] Unresolved assumptions are visible
- [ ] Recommendation type is valid when merge-related
