# Semiformal Reviewer Guide

Reviewers should validate the evidence spine, not prose volume.

## Fast review sequence

1. Confirm task statement and decision scope.
2. Verify material claims have evidence locations.
3. Verify alternative hypothesis (or counterexample pressure) exists.
4. Verify unresolved assumptions remain visible.
5. Verify final recommendation is derivable from artifact body.

## Required reviewer comment pattern

- strongest evidence in favor:
- strongest unresolved concern:
- counterexample status:
- merge recommendation:
- what would change the decision fastest:

## Recommendation outcomes

- safe-to-merge
- safe-with-conditions
- not-yet-safe
