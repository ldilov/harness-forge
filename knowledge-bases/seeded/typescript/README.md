# TypeScript Knowledge Base

    TypeScript knowledge base for Node services, React/Next frontends, libraries, CLIs, and monorepos.
Use this pack to deepen rules around typing, runtime validation, testing, security, and framework-specific examples.

    ## What is included

    - `rules/common/` baseline rules copied from the current package
    - `rules/typescript/` starter language pack for the fresh project
    - `docs/` notes for tooling, frameworks, examples, and review checklists
    - `examples/` scenario seeds for expanding future project examples

    ## Recommended use

    1. Use the `rules/` files as the starter pack for the fresh project.
    2. Pull examples into the public docs and scenario examples.
    3. Extend the review checklist as agent or command quality gates.
    4. Split framework-heavy content into separate docs if the pack grows too large.
