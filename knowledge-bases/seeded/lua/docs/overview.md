# Lua Overview

    ## Best fit

    Lua knowledge base for general scripting, Neovim plugins, OpenResty services, game scripting, and embedded automation.
Use this pack to add first-class Lua rules, examples, and harness workflows.

    ## Recommended tooling

    - Lua
- LuaRocks
- stylua
- luacheck
- busted
- Plenary test harness for Neovim

## Framework and ecosystem areas

    - Neovim plugin development
- OpenResty
- LÖVE/game scripting
- embedded host integrations

## Common pitfalls

    - global state leakage
- nil handling surprises
- metatable overuse
- unsafe shell invocation
- weak module boundaries
