# Java Overview

    ## Best fit

    Java knowledge base for service applications, libraries, Spring Boot APIs, message-driven systems, and test-heavy enterprise backends.
Use this pack to deepen rules, examples, hooks, and review checklists.

    ## Recommended tooling

    - JDK
- Gradle
- Maven
- JUnit 5
- AssertJ
- Mockito
- Testcontainers
- Spotless
- google-java-format

## Framework and ecosystem areas

    - Spring Boot
- Jakarta
- Micrometer
- Testcontainers
- Gradle multi-module builds

## Common pitfalls

    - field injection
- framework leakage into domain
- overly generic exceptions
- hidden N+1 queries
- too much mutable shared state
